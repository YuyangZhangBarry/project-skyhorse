# Project Skyhorse - 天马行空问答

一款收录数百个富有想象力问题的问答 App。用户浏览问题后，通过选择题或简答题作答，后台 AI 对简答内容进行智能评分并给出反馈。

## Tech Stack

- **Client**: Flutter (iOS + Android)
- **Backend**: Python + FastAPI
- **Database**: PostgreSQL + Redis
- **AI Scoring**: DeepSeek (primary) / OpenAI (fallback)
- **Payments**: RevenueCat (App Store / Google Play IAP)

## Project Structure

```
project-skyhorse/
├── app/          # Flutter client
├── backend/      # Python FastAPI backend
└── docs/         # Documentation
```
